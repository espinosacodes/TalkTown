# TalkTown

AI-powered language learning RPG built at **Zero to Agent: Cali** — Vercel's global build week.

## About the Event

**Zero to Agent: Cali** | April 24, 2025 | AceSpace Creative Suite, Cali, Colombia

Part of Vercel's global **Zero to Agent** build week (April 24 – May 3), this event brought together builders, creators, and founders to build and deploy real AI agents in a single day. All projects were submitted to the global Zero to Agent Competition with over $6,000 in prizes.

Hosted by [AceSpace](https://lu.ma/zerotoagent-cali) — a creative suite home to content creators, influencers, personal brands, and founders.

## What is TalkTown?

TalkTown is an interactive language learning RPG where players explore a virtual town, interact with AI-powered NPCs, and practice languages through immersive dialogue. Each NPC has a unique personality and responds dynamically using AI, making every conversation a learning opportunity.

## Tech Stack

- **Framework:** Next.js 16 + React 19
- **AI:** Vercel AI SDK (Google & Groq providers), Vercel Workflows
- **Auth:** Name-based (localStorage)
- **Database:** AWS DynamoDB
- **Styling:** Tailwind CSS 4 + Radix UI
- **Language:** TypeScript

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
